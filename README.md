# 💰 Expense Tracker

A modern full-stack MERN Expense Tracker application with JWT Authentication, Dashboard Analytics, Transaction Management, Excel Export, User Profile, Charts, Search & Filter, and Responsive UI.

---

## 🚀 Features

### 🔐 Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Secure Password Hashing

### 📊 Dashboard
- Total Balance Overview
- Income Analytics
- Expense Analytics
- Interactive Charts
- Recent Transactions Widget

### 💳 Transaction Management
- Add Transactions
- Edit Transactions
- Delete Transactions
- Search Transactions
- Filter by Category
- Pagination
- Export Transactions to Excel

### 👤 User Profile
- User Information
- Dynamic Statistics
- Total Savings
- Total Income
- Total Expenses
- Transaction Count
- Joined Date

### 🎨 UI Features
- Responsive Design
- Modern Dark Theme
- Toast Notifications
- Clean Dashboard Layout
- React Icons Integration

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Vayu-143/expense-tracker.git

cd expense-tracker
```

---

# Backend Setup

```bash
cd server

npm install
```

Create `.env`

```env
MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

Run Backend

```bash
npm run server
```

---

# Frontend Setup

```bash
cd client

npm install

npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server folder:

```env
MONGO_URI=your_mongodb_uri

JWT_SECRET=your_secret_key
```
